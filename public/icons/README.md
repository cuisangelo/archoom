# Cloud provider icons

The SVGs in this directory are the official architecture icon sets published by
each cloud provider, imported verbatim by `scripts/import-cloud-icons.mjs`:

- **aws/** — [AWS Architecture Icons](https://aws.amazon.com/architecture/icons/) (package 2026-04-30)
- **azure/** — [Azure architecture icons](https://learn.microsoft.com/en-us/azure/architecture/icons/) (V23)
- **gcp/** — [Google Cloud icon library](https://cloud.google.com/icons)

These icons are **not** covered by this repository's license. They remain the
property of Amazon Web Services, Microsoft, and Google respectively, and are
redistributed here solely for use in architecture diagrams as permitted by each
provider's icon terms (linked above). Do not crop, distort, recolor, or use
them to represent non-provider products.
